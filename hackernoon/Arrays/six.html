6. How is an integer array sorted in place using the quicksort algorithm

Solution

I picked the pivot as the first number in the subset for simplicity sake. This way whenever I found a new number lower than the pivot, all I had to do was swap the pivot with the number to it's right (after switching that with the number found)

Other options would be to pick the pivot on the median etc - at the cost of having to find that first. Though this could be found as you do the comparisons. You still have to compare to every item but this would allow for therotically less swaps. And therefore cutting off sections of the array pretty quickly.
